# Cursor AI 配置信息

## 開發環境配置

### WP-CLI 路徑
- **WP-CLI 安裝位置**: `E:\WP-CLI`
- **WP-CLI phar 文件**: `/e/WP-CLI/wp-cli.phar`
- **使用方式**: 必須通過 XAMPP 的 PHP 執行（見下方說明）

### 項目路徑
- **項目根目錄**: `D:\XAMPP\htdocs\wordpress\wp-content\plugins\mp-ukagaka`

## 常用命令

### 生成翻譯文件
```bash
cd "D:\XAMPP\htdocs\wordpress\wp-content\plugins\mp-ukagaka"
# 使用 XAMPP 的 PHP 執行 WP-CLI
/d/XAMPP/php/php.exe /e/WP-CLI/wp-cli.phar i18n make-pot . languages/mp-ukagaka.pot --domain=mp-ukagaka
```

### 編譯翻譯文件
```bash
cd "D:\XAMPP\htdocs\wordpress\wp-content\plugins\mp-ukagaka\languages"
python compile_po.py
```

### PHP 路徑（XAMPP）
- **XAMPP PHP 位置**: `D:\XAMPP\php\php.exe`
- **Git Bash 路徑**: `/d/XAMPP/php/php.exe`
- **使用方式**: WP-CLI 需要通過 XAMPP 的 PHP 執行

## 注意事項

- WP-CLI 需要使用完整路徑執行，或確保已添加到 PATH
- **使用 XAMPP 的 PHP**: WP-CLI 需要通過 XAMPP 的 PHP (`D:\XAMPP\php\php.exe`) 來執行
- Python 版本: 3.10.6
- 翻譯文件編譯使用 `compile_po.py` 腳本

## WP-CLI 使用說明

### 使用 XAMPP PHP 執行 WP-CLI（推薦方法）
```bash
# 直接通過 XAMPP 的 PHP 執行 WP-CLI phar 文件（此方法已測試成功）
/d/XAMPP/php/php.exe /e/WP-CLI/wp-cli.phar [命令]

# 例如：
/d/XAMPP/php/php.exe /e/WP-CLI/wp-cli.phar --version
/d/XAMPP/php/php.exe /e/WP-CLI/wp-cli.phar i18n make-pot . languages/mp-ukagaka.pot --domain=mp-ukagaka
```

**注意**: `/e/WP-CLI/wp` 腳本需要系統 PHP，無法使用，請直接使用 phar 文件。
